Detailed Description

Preform facet region growing on weed point cloud to get "facets". Ideally facets represent full leaves, but we use the term "facets" for generality Facet-region growing method presented here: https://www.ncbi.nlm.nih.gov/pmc/articles/PMC6263610/ All code is adapted from psuedocode presented in the paper.

Algorithm 2A: Seed point selection and coarse planar facet generation.

Generates (extremely) coarse facets on the leaves from chosen set of seed points.

Parameters
pointsa list of points from the point cloud from the selection
characteristicsthe list containing normals and smoothnesses as output from the spatial_characteristics function
Kinitial number of points in the neighborhood, tunable
grow_threshthe euclidean distance between a candidate grow point and the seed point must be less than this number (denoted as r_1 in the paper)
angle_threshthe angle between the normals of the fitted planes of the seed point and the current points needs to be less than this number (denoted as theta in the paper)
plane_threshthe distance between the point and the fitted plane of the seed point needs to be less than this number (denoted as sigma_1 in the paper)
Returns
the list of seed points, the list of labels for each point in points (assigning it to the facet associated with each seed point)

Algorithm 2: Facet over-segmentation.

Calls coarse_segmentation and kmeans_refinement in succession and returns the labels. Currently skipping K-Means facet refinement because it works better for some reason.

Parameters

characteristicsthe list containing normals and smoothnesses as output from the spatial_characteristics function
Kinitial number of points in the neighborhood, tunable
grow_threshthe euclidean distance between a candidate grow point and the seed point must be less than this number (denoted as r_1 in the paper)
angle_threshthe angle between the normals of the fitted planes of the seed point and the current points needs to be less than this number (denoted as theta in the paper)
plane_threshthe distance between the point and the fitted plane of the seed point needs to be less than this number (denoted as sigma_1 in the paper)
seed_pointsthe list of seed points from coarse_segmentation
sphere_radiusthe radius of the sphere containing the points around each seed point to consider for reassignment of labels (denoted as r_2 in the paper)
Returns
the list of labels for each point in points (assigning it to a facet)

Algorithm 3: Facet region growing.

Combines facets into larger facets based on distance and adjacency.

Parameters
facetsa list of points from the point cloud from the selection
grow_threshthe distance from the center of facet j to facet i must be less than this number (denoted as sigma_2 in the paper)
adj_threshthe smallest distance between two points in two facets must be less than this for the facets to be considered adjacent
Returns
the list of facets (excluding normals because they are no longer needed)
