Detailed Description

Preform facet region growing on weed point cloud to get "facets". Ideally facets represent full leaves, but we use the term "facets" for generality Facet-region growing method presented here: https://www.ncbi.nlm.nih.gov/pmc/articles/PMC6263610/ All code is adapted from psuedocode presented in the paper.
