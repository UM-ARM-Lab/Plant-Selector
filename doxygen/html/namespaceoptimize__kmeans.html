Detailed Description

Uses training data to generate weights for kmeans distance function that will create "optimal" solutions.

Evaluate the cost given the weights.

Uses the weights to do initial segmentation on the training point clouds and then finds the average distance between the manual labels and the centroids of the segmented clouds.

Parameters
weightsa list of 6 weights, corresponding to x, y, z, r, g, and b
training_directorya string with the directory to the training data
pc_filenamesa list of strings containing the names of the pointclouds being used for training. eg: "79.ply"
manual_label_filenamesa list of strings containing the names of the manual labels for each pointcloud in pc_filenames. eg. "79.npy"
Returns
the cost
